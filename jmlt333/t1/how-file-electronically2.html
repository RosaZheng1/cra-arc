Steps

  1. Begin filing your taxes

    Be prepared to file by:

    • Knowing important dates
    • Having the tax slips
    • Having updated personal information

    For details, refer to: Get ready to file a tax return

    Filing someone else's tax return

    If you are filing a personal income tax return on behalf of someone else, you must be authorized as their representative.

    For more information, refer to: Representative authorization

  2. Choose the software

    Select from the list of CRA's certified tax software to file the personal income tax return.

    Refer to: Certified tax software

  3. Complete the tax return

    1. Enter personal information

      Enter the current personal information, such as name, address, and SIN.

      Access code

      Entering the access code is optional, but recommended.

      Tax software can still send your return even if you don't provide the access code.

      Where to find your access code

      Your 8-character access code is made up of numbers and letters and is located on the right side of your Notice of Assessment for the previous tax year.

      If you are filing a tax return for the first time ever, an access code is not needed.

      What happens if you don't provide your access code

      While this access code is not mandatory, if you do not enter your access code:

      • You will not be able to use any information from your 2023 tax return when confirming your identity with the CRA
      • You will have to rely on other information for authentication purposes

      Use the "Auto-fill my return" service

      "Auto-fill my return" is an online service where your tax slips and other personal information can be securely imported from the CRA to your tax software.

      Refer to: Using the "Auto-fill my return" service to fill in a tax return

    2. Report income

      What you need to report as income, including self-employment income, foreign investment income, income from assets, or other earnings.

      Refer to: Income that must be reported

    3. Claim deductions, credits, and expenses

      Find and claim tax deductions, credits, and expenses to reduce the amount of tax you owe at the end of the tax year.

      Refer to: Deductions, credits, and expenses that can be claimed

    4. Review the information on the tax return

      Before sending your tax return to the CRA, it is your responsibility to review the information to make sure it is accurate.

      Transferring amounts to an instalment account

      If you expect a refund and want to have it transferred to an instalment account, select this option when you fill in your tax return online.

      The CRA will transfer the full amount of your refund directly to your instalment account. It will go toward any amount owing on your tax return. The CRA will consider this payment received on the date that your return is assessed.

      For more information, refer to: Required tax instalments for individuals

  4. Send the tax return to the CRA

    Use the "NETFILE" service to send the return

    "NETFILE" is an online service where you are able to send tax returns directly to the CRA. You must make sure you are eligible to use the service.

    Refer to: Using the "NETFILE" service to send a tax return

    • Eligible to use the service

      If you are eligible to use the "NETFILE" service, you can send the digital version of your return to the CRA and within a few seconds you will receive a confirmation code.

    • Not eligible to use the service

      If you are not eligible to use the "NETFILE" service, or if you choose to print your return and send it, refer to the method of sending a paper return to the CRA.

    Use the "Express NOA" service to verify the return

    "Express NOA" is an online service where you can view an express version of your notice of assessment (NOA), immediately after you file.

    Refer to: Using the "Express NOA" service after sending a tax return

  5. Proceed with finishing your taxes

    There are 3 possible outcomes from your tax return:

    • Get a refund
    • Have a balance owing
    • Have no refund or balance owing

    You will receive your notice of assessment (NOA) after your tax return is processed. It will indicate the outcome from your tax return.

    If you need to make changes to your tax return, you must wait until you receive your NOA.

    Processing time: Within 2 weeks

    For details, refer to: After filing a tax return

Using an automated service to fill in your tax return

Auto-fill my return is an online service that allows individuals and authorized representatives using certified tax software, to automatically fill in parts of a personal tax return with information that the CRA has available at the time of the request.

How to use this service

Individuals and authorized representatives can use Auto-fill my return.

Individuals

To use the Auto-fill my return service, you must:

  • Be registered for CRA's My Account

    Refer to: CRA My Account

  • Use a certified tax software

    Refer to: Certified tax software (NETFILE)

If you prepare a tax return for a family member or friend, you need to be an authorized representative.

Authorized representatives
If you are a family member or friend
If you are a professional service provider (such as an accountant or discounter)

To use the Auto-fill my return service, you must:

  • Be a certified electronic filer

    Refer to: Apply for EFILE

  • Be registered in Represent a Client

    Refer to: Represent a Client

  • Use a certified tax software

    Refer to: Certified tax software (EFILE)

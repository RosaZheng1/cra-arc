NETFILE-certified tax software

Steps

  1. Determine how you are using the software

    • To fill in the return

      Select the CRA certified tax software you would like to use to fill in the tax return.

    • To send the return

      Make sure you are eligible to send the tax return using the NETFILE service.

      (Most taxpayers are eligible)

  2. Select the software

    Many software products will prepare the tax return for free if you have a modest income with a basic return, otherwise there is usually a small service fee.

Paying the balance owing on a personal income tax

If you have an amount on line 48500 of your personal income tax return, you have a balance owing.

Steps

  1. Determine how you are using the software

    • Fill in the return

      Select the CRA certified tax software you would like to use to fill in the tax return.

    • Send the return

      Determine if you are eligible to send the tax return using the NETFILE service.

  2. Select the software

  3. Research the software

    Tax software companies are not representatives of the CRA.

    Keep informed that:

    • The CRA does not look at the privacy policies of software developers.

      It is your responsiblity to research the policies before buying or using tax software.

    • The use of the software is the responsibility of the software developer.

      The CRA cannot be held responsible if programming errors affect the calculation of amounts payable.

How to file a personal income tax return for someone else

Intro text

Steps

  1. Determine how you will fill in the tax return

    If you are a family member or friend

    • Online

      Get the tax software to fill in the tax return.

      Refer to: Certified tax software

    • By paper

      Get the paperwork to fill in the tax return.

      Refer to: Get a T1 income tax package

    If you are a professional service provider

    Online

    If you are a professional service provider, such as an accountant or discounter, you must determine if you are eligible to fill in the tax return using certified tax software, and review the softwares that are available.

    Refer to: EFILE for individuals

  2. Complete the authorization process

    The person you are doing taxes for must authorize you for access to their CRA tax information, whether you are filing online or by paper.

    Once you are authorized, you can file their tax return for them.

    For more information, refer to: Authorize a representative

  3. Fill in the tax return

    1. Enter the personal information

      Enter the current personal information of the person you are preparing the tax return for, such as their name, address and SIN.

      Getting the NETFILE access code

      If you are filing online, entering the access code is optional, but recommended.

      Tax software can still elecrontically transmit the return using the CRA's secure NETFILE service, even if you don't provide the access code.

      Where to find the access code

      The 8-character access code is made up of numbers and letters and is located on the right side of the Notice of Assessment for the previous tax year.

      If you are filing a tax return for the first time ever, an access code is not needed.

      What happens if you don't provide the access code

      While this access code is not mandatory, if you do not enter your access code:

      • You will not be able to use any information from the 2023 tax return when confirming your identity with the CRA
      • You will have to rely on other information for authentication purposes
      Using Auto-fill my return

      If you are filing online, tax slips and other personal information can be securely imported from the CRA to your tax software.

      For more information, refer to: Using an automated service to fill in a tax return

    2. Report the income

      What you need to report as income, including self-employment income, foreign investment income, income from assets, or other earnings.

      If you need a copy of your tax slips

      With NETFILE, you don't need to send your tax information slips, receipts, or other documents used to prepare your tax return unless the CRA asks to see them.

      If you are filing by paper and are missing tax slips, refer to: [Tax slips]

      It is your responsibility to contact the issuer for any missing or incorrect tax slips.

    3. Claim the deductions, credits, and expenses

      Find and claim tax deductions, credits, and expenses to reduce the amount of tax you owe at the end of the tax year.

    4. Review the information on the tax return

      Before sending the tax return to the CRA, it is your responsibility to review the information to make sure it is accurate.

      Transferring amounts to an instalment account

      If you expect a refund and the person you are preparing the tax return for wants to have it transferred to an instalment account, select this option when you fill in the tax return online.

      The CRA will transfer the full amount of the refund directly to the instalment account. It will go toward any amount owing on the tax return. The CRA will consider this payment received on the date that the return is assessed.

      For more information, refer to: Required tax instalments for individuals

  4. Send the tax return to the CRA

    If you completed the return online
    If you are a family member or friend

    If you want to send the return online, you must make sure you are eligible to use the NETFILE service.

    Refer to: Using the NETFILE service

    If you are not eligible to use the NETFILE service, or if you choose to print your return and send it, refer to sending the return by mail.

    Sending the return using the NETFILE service

    Processing time: Within 2 weeks

    Send the digital version of your return to the CRA and within a few seconds you will receive a confirmation code.

    You may also receive your Express NOA.

    If you are a professional service provider (such as an accountant or discounter)

    Determine if you are eligible to send the tax return using certified tax software.

    Refer to: EFILE for individuals

    If you completed the return by paper

    Sending the return by mail

    Processing time: Within 8 weeks

    If there are multiple returns being filed

    If there are multiple returns being filed:

    • Place each person's return in a separate envelope
    • Place the same person's returns all in one envelope
    Transferring amounts to an instalment account

    If you expect a refund and want to have it transferred to an instalment account, you must attach a note to your paper return.

    The CRA will transfer the full amount of your refund directly to your instalment account. It will go toward any amount owing on your tax return. The CRA will consider this payment received on the date that your return is assessed.

    For more information, refer to: Required tax instalments for individuals

    • Mailing to the CRA

      1. Find out where to mail a paper return
      2. Mail your completed tax return to your tax centre

    • Dropping off at a CRA office

      1. Find out the location of CRA's drop boxes
      2. Place your return in a CRA drop box

  5. Proceed with finishing the taxes

    There are 3 possible outcomes from the tax return:

    • Get a refund
    • Have a balance owing
    • Have no refund or balance owing

    The person you are doing taxes for will receive their notice of assessment (NOA) after the tax return is processed. It will indicate the outcome from their tax return.

    If you need to make changes to the tax return, you must wait until they receive their NOA.

    • If you filed online

      • Processing time: Within 2 weeks

    • If you filed by paper

      • Processing time: Within 8 weeks

    For details, refer to: After filing a tax return

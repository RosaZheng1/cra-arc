Using certified tax software to do your taxes (NETFILE)

NETFILE is an online service that allows individuals and tax preparers send personal tax returns directly to the Canada Revenue Agency (CRA) using certified tax software.

The NETFILE service is open for electronic filing from February 19, 2024 until January 24, 2025. It is available 21 hours a day. Due to maintenance activities, the service is not available from 3am to 6am (Eastern time).

    Most taxpayers can use NETFILE to complete and send their personal tax returns, however you are not able to use NETFILE to apply for benefits or credits or update your personal information.

    1. Research your software

      Tax software companies whose products are certified for NETFILE are not representatives of the CRA. The CRA does not look at the privacy policies of software developers. It is your responsibility to research these policies before buying or using a software product or web application.

      Use of the software is the responsibility of the software developer. The CRA cannot be held responsible if programming errors affect the calculation of income tax and benefits payable.

    2. Select your software

      Select the CRA certified tax software you would like to use to file returns from 2017 and later. You will be re-directed to the software developer's web page, where you can access software products that use the CRA's NETFILE service.

      After you select your software, you will be able to use other CRA services such as:

      • Auto-fill my return
      • Express NOA
      • ReFILE

  3. Using this service

    Tax software must be certified by NETFILE every year. Review our NETFILE certified tax software lists to confirm that your software is certified for the year you need.

    1. Know the limitations

      If you are filing a Quebec tax return

      With the NETFILE service, all provincial and territorial tax returns are filed as schedules to your federal income tax return, except returns for Quebec.

      To learn more about filing your Quebec provincial tax return electronically, refer to: Revenu Québec

      If you are filing multiple returns

      If you are a tax preparer registered in Represent a Client (RAC) to act on behalf of others, you may file up to 20 returns per computer or online account for each tax year.

      Tax software companies must respect this limit to be certified for CRA's NETFILE service.

    2. Use your NETFILE access code

      Entering the access code is optional, but recommended.

      Tax software can still elecrontically transmit your return using the CRA's secure NETFILE service, even if you don't provide the access code.

      Where to find your access code

      Your 8-character access code is made up of numbers and letters and is located on the right side of your Notice of Assessment for the previous tax year.

      If you are filing a tax return for the first time ever, an access code is not needed.

      What happens if you don't provide your access code

      While this access code is not mandatory, if you do not enter your access code:

      • You will not be able to use any information from your 2023 tax return when confirming your identity with the CRA
      • You will have to rely on other information for authentication purposes

    3. Keep your software updated

      If you are using uncertified software or an older version of the software, you may need to update or download a certified version from the developer. Information on how to update your software is available on the developer's website or from within the software itself.

      In some cases not updating your software will prevent you from being able to use NETFILE.

    4. Have confidence in the service

      The CRA is responsible for making sure personal and financial information is sent in an encrypted format between your computer and our servers. We are also responsible for the security and confidentiality of your information once we receive it, and take precautions to ensure that there is no unauthorized access to your data.

      When using this service, the CRA performs several identification validations with the information we have in our database. If it finds an identification error or a discrepancy, the system will automatically stop processing your return.

Change a personal income tax return using the "ReFILE" service

You may be looking for:

  • Change a personal tax return using a CRA secure portal

"ReFILE" is an online service that allows individuals and tax preparers send online adjustment requests for personal tax returns using certified tax software.

The ReFILE service is:

  • Open February 19, 2024 until January 24, 2025
  • Available 21 hours a day, from 6am to 3am (Eastern time)
  • Closed for maintenane from 3am to 6am (Eastern time)

Steps

  1. Wait for your tax return to be processed

    You must wait until you receive your notice of assessment (NOA) before you can request to make changes to your personal income tax return.

  2. Ensure you used certified tax software

    You or your tax preparer must have used certified tax software to file the return you are requesting to change.

    When changing your own tax return

    You must have used NETFILE certified tax software

    When changing someone else's tax return

    • If you are a family member or friend, you must have used NETFILE certified tax software

    • If you are a professional service provider (such as an accountant or discounter), you must have used EFILE certified tax software

  3. Determine if you can use this service

    Answer interactive questions

    A text version is available if needed.

    You are not able to use ReFILE to apply for benefits or credits.

    If the interactive questions do not appear, view the text version.

    • An individual changing your own tax return
    • A tax preparer changing someone else's tax return

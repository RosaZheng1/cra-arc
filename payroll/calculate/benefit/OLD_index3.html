Situation: Only remuneration provided is a taxable benefit

Do not deduct CPP contributions if all of the following apply:

  • It is a non-cash taxable benefit (with the exception of security options)
  • It is the only remuneration you provide to your employee (your employee did not receive any cash earnings)

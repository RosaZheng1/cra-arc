Quotations – Standard web pages

Quotations are used to cite a group of words by someone other than the original author or speaker.

Correct use

  • Use blockquotes to indicate a large quotation of text that comes from another source
  • Name the quote source when possible, including the author's name and source title
  • Capitalize the cited source title the same as the author does

Incorrect use

  • Do not add quotation marks (") or images of quotation marks to the text, as this is redundant styling
  • Do not use if you can link directly to the source content
  • Do not use if you can rephrase the content into concise, plain language
  • Do not use to format content that is not a quote
  • Do not use italics

Inline quotes

Use inline quotes for snippets of content that are quoted from another source. They appear within a sentence.

Appearance

Complete sentence with colon: "Quoted sentence!"

Someone states or intro phrase, "section of quoted text."

Introductory phrase, "section of quoted text that now needs a comma," more text.

HTML code 
<!-- Complete sentence with colon, then quote -->
<p>Complete sentence with colon: <q>"Quoted sentence!"</q></p>

<!-- Someone states or intro phrase, then quote -->
<p>Someone states or intro phrase, <q>"section of quoted text."</q></p>

<!-- Someone states or intro phrase, quote that now needs a comma -->
<p>Introductory phrase, <q>"section of quoted text that now needs a comma,"</q> more text.</p>

Coding considerations:

  • Wrap the quote, including quotation marks inside a <q> element
  • Place periods and commas inside the closing </q>
    • This is the style CRA uses for quotation marks
    • Unless they are part of the original quotation, place all other marks outside the quotation marks

Correct use

  • Use inline quotes if the quote is less than a few words in length
  • Use double quotation marks (") for inline quotes

Incorrect use

  • Do not use if the content can be rephrased into concise, plain language

Quotation marks

Use to identify a title, button, or action that is not a link.

Appearance

Then select the "Sign in" button.

HTML code 
<p>Then select the "Sign in" button.</p>

Correct use

  • Use quotation marks to identify a title, button, or action that is not around or in a link

Incorrect use

  • Do not use quotations around a link
  • Do not use if the content can be rephrased into concise, plain language

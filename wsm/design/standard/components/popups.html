Coding considerations:

  • Provide a way of opening the overlay:
    • Link: Add a link to the page where the href and the aria-controls attribute match the id attribute on the section element of the overlay
    • Event: Trigger the overlay with the following JavaScript code: 
      $( "#unique-id" ).trigger( "open.wb-overlay" );

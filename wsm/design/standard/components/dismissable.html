Dismissable content – Standard web pages

Use to make content dismissable by users.

On this page

  • How to implement
  • Complementary components and functions

How to implement

Add the class wb-dismissable to the element that contains the content you want to make dismissable.

The content can be closed by the user (x), and won't be visible when they return to visit the page.

Appearance

Dismissable content

The content in the following alert is dismissable. If you close it by clicking the x on the top right corner, it will not be visible for you when you refresh or reopen the page.

The content inside this box is dismissable.

Bring dismissable content back, by clicking on the button below (demo only). You must refresh the page to see the effect of this button.

HTML code 
<section class="wb-dismissable">

</div>

Correct use

  • Use when you want to allow the user to dismiss non-primary content that they only need to see once (e.g., advertisements, notices)

Incorrect use

  • Do not use when the content is primary content

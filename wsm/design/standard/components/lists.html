How to use

A short vertical list is easier for people to scan and remember than a long paragraph. If possible, aim for maximum of 7 items in your list.

When writing lists

  • arrange list items in:
    • logical or sequential order, or by
    • priority, importance, or popularity, or by
    • alphabetical, on rare occasions
  • use positive statements as much as possible
  • use one sentence per single list item
  • place negatively phrased items together, if you must use them
  • use similar text and list types consistently within the same document
  • use consistent grammatical structure. For example, if you:
    • use the imperative mood (or command) for the first item of your list, use the same mood for each subsequent list item
    • start your first item with a noun, start the other list items with nouns
  • if a list contains more than 7 items, consider breaking it up into categories

Capitalization

When list items are grammatically independent of each other or follow a heading, capitalize the first letter of each list item.

Example - punctuation in independent list items

Most requested

  • Financing your new business
  • Financing for innovation
  • Find a loan for your small business
  • Canada job grants for employers

When the meaning of list items depends on a lead-in phrase, don't capitalize the first letter of each unless it's a proper name.

Example - punctuation in interdependent list items preceded by a colon

Representatives can be:

  • citizenship or immigration consultants
  • lawyers
  • other representatives

Punctuation

Don't end the list items with any punctuation.

Lead-in to a list

Each list requires one of the following:

  • a descriptive and unique heading
  • a lead-in sentence
  • an introductory paragraph to provide some context

If you need to clarify the relationship between each element, start the list with a lead-in paragraph that:

  • introduces or applies to all the list items
  • emphasizes the common element between all the items

If necessary, specify "all" or "or" in the introductory paragraph to clarify whether the list is comprehensive or conditional.

Example - lead-in sentence to a list

You can write:

  • "You must meet all of the following requirements:"
  • "To be eligible, you must meet 1 or more of the following requirements:"

Example - bulleted list items

Write
To draft a high-quality source document:

  • organize your ideas
    • arrange your ideas from most to least important to the task
    • test your order with a representative audience
  • use the active voice
  • use the right words
  • write short sentences and paragraphs
  • follow the guidelines in the Canada.ca Content Style Guide

Instead of
To draft a high-quality source document:

  • organize your ideas
  • arrange your ideas from most to least important to the task
  • test your order with a representative audience
  • do not use the passive voice
  • it is important to use the right words
  • use only short sentences and paragraphs

Correct use

Numbered lists
  • Use ordered lists to show:
    • ranking
    • order
    • priority
    • step-by-step instructions
Alpha lists
  • Use to display letters instead of numbers
    • This applies only to immediate children list items
    • This means you also need to add the CSS class for any nested lists
  • Use lower alpha mainly as a nested second level list
  • Use upper alpha mainly in legal documents
Roman numeral
  • Use to display Roman (I, II, III, IV or i, ii, iii, iv…) instead of Arabic (1, 2, 3, 4…) numerals
    • This applies only to immediate children list items
    • This means you also need to add the CSS class for any nested lists
  • Use lower roman mainly as a nested second level list
  • Use upper roman mainly in legal documents

Correct use

  • Use to make a comparison between yes/no, can/cannot or correct/incorrect
  • Use as an explicit statement of when you can/must do something
  • Use with a clear call to action when a list of steps must be taken
  • Add X marks for the actions athat should not be considered or to say what not to do
  • Use as a list of specific things something (a rule) applies to
  • If there are sub bullets that don't have a call to action the header should have the checkmark, as long as the header has an action or a clear result for the user

Incorrect use

  • Do not use in a list explainig contents
  • Do not use for actions that you might do
  • Do not use if there is no clear call to action
  • Do not use outside of a list (paragraph)
  • Do not use in conjunction with another graphic or image
  • Do not use as a header, with bullets included beneath
  • Do not use for theoretical information

Supporting research

Home office expenses - User testing

The checkmarks and X's were initially tested as part of the new Home Office Expenses web pages. Due to COVID-19, CRA knew that hundreds of thousands of Canadians would suddenly become eligible to claim their work space in the home expenses when previously they would not have had this experience.

The web pages were entirely re-done in order to make it as easy as possible to understand what taxpayers could claim for their home office, without calling the call centre.

Here, the checkmarks and X's treatment was used to show the expenses which could and could not be claimed depending on the type of employee the taxpayer is.

Rather than sorting through content and reading paragraphs of text (as they had done previously), people could easily read a clear, bulleted list of common expense claims. Users immediately understood that if there was a checkmark next to the expense, it meant they could claim it, and if there was an X, it meant they could not.

This formatting made the content more scannable and easier to understand.

The first user test we conducted with the checkmarks and X's resulted in 7/8 participants getting the correct answer. User 3 said "I like the face that they are all displayed in this manner. [It] makes browsing simple".

Visit Home office expenses for employees - What the changes are to see a real life example of checkmarks and X's.

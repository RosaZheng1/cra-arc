Charts and graphs – Standard web pages

Use to dynamically generate charts and graphs from table data.

On this page

  • When to use
  • Working examples
    • Bar chart
    • Pie chart
  • How to implement
  • Complementary components and functions

When to use

Use to to publish simple or complex charts online, before using an image-based charts approach. This will save you time in creating long text descriptions for your charts and will create a uniform charting style accross your website.

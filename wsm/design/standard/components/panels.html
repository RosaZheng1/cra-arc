Panels – Standard web pages

Use to spotlight general information/actions within a self-contained box. Panels have colour themed options, as well as offer a distinct header and footer.

Coding considerations:

  • Ensure a heading (<h*>) is used within the panel heading
  • Use .panel for both full-width content, as well as content that is padded in from the border with .panel-body
  • Include a <caption> when the <table> is not directly under an <h*> tag
  • Use .wb-inv when it is necessary to make the caption invisible, this ensures accessibility

Use the correct type of panel

  • Use info to spotlight specialized information/actions
  • Use primary to spotlight general information/actions
  • Use success to spotlight correct information/actions
  • Use warning to spotlight warning information/actions
  • Use danger to spotlight dangerous information/actions

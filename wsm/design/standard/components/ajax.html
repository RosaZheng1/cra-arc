Data Ajax – Standard web pages

Use Ajax calls to load sections of content dynamically from a different webpage.

On this page

  • When to use
  • How to implement
  • Complementary components and functions

When to use

Correct use

  • Use to reproduce frequently updated content in one or more areas (e.g. news items)
  • Use to provide a snapshot of part of another page to focus on a specific area (e.g. a definition item from a glossary)
  • Use sparingly and effectively. If the source page is deleted or moved, there is no visual indication on the page with the Ajax code (i.e. no 404 error message). When in doubt use a hyperlink instead

Incorrect use

  • Do not pull content from another Branch/Region without the explicit approval of the source material's owner
  • Do not pull the entire content of a webpage, or content from outside Canada.ca

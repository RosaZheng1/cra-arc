Law of common fate

On this page

  • Overview
  • Examples
  • Take aways

Overview

Definition: We perceive objects as lines that move along a path. We group them together when they have the same direction of motion.

In practice: Objects with a path of movement are interpreted as one unified group. This can be used to guide the user’s attention.

Examples

Correct use

  • The following example uses the law of common fate, as the faceted search options form a vertical line down the right side of the page, uniting a group of filters that can be used to refine search results.
  • In the next page, the previous and next buttons horizontally align and can be understood as parts of the same navigation mechanism.
  • The gray background colour of horizontal table headers in the next example delineates the headers from the contents and unifies them.

Incorrect use

  • The majority of the table data in the next page has the same dark gray background colour, both horizontally and vertically, which negates any value that might have been intended.

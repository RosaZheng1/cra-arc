Sections

Principle of visibility
Users need to easily find and see ways to use the page
Principle of feedback
Users need to be informed of the outcome of their action immediately after it occurs
Principle of affordances
The visual design properties of an object should suggest how to use it
Principle of mapping
There should be a relationship between controls and their effect
Principle of constraints
There should be restrictions built into the interaction to reduce user error
Principle of consistency
Similar actions and similar elements should be used to accomplish similar tasks (this reduces error and improves learnability)

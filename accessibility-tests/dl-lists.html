Description list aria role

Situation

Issue: most screen readers read dl lists as a list with items that include the term <dt> and the <dd> as seperate items.

Goal: create different screen reader outputs based on need.

All items are read in Chrome, may vary based on browser and OS. JAWS inspect is not a replacement for JAWS and should be validated.

Default

Base level HTML with custom CSS class
